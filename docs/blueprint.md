# **App Name**: Excorx Bonus Hub

## Core Features:

- Offer Display UI: Render a visually appealing gift offer card, detailing the 200 USDT reward and its conditions, replicating the provided HTML/CSS.
- User Credential Input: Provide form fields for users to securely enter their Excorx-linked email and password.
- Form Submission Handling: Manage the client-side validation and state (button disabling, text change) during the submission process.
- Simulated Gift Request API: Implement a Next.js API route that simulates the backend processing of the gift request, mimicking the success response of the original save_gift.php, without actual database interaction as per user request.
- Submission Review Notification: Display a success message indicating that the request is under review upon successful (simulated) submission.

## Style Guidelines:

- Color Scheme: Dark scheme to convey a premium, digital finance feel.
- Primary color: A vibrant blue, '#4BB2FF' (HSL: 215, 90%, 65%), for interactive elements and emphasis, symbolizing trust and technology in the digital financial space. This is intentionally bright to stand out against the dark background.
- Background color: A very dark, subtly blue-tinged grey, '#192126' (HSL: 215, 20%, 12%), creating a deep and immersive backdrop that allows UI elements to pop.
- Accent color: A bright, airy cyan, '#9EEBFF' (HSL: 185, 70%, 70%), derived as an analogous hue to the primary. This adds a refreshing highlight without disrupting the core palette's harmony, useful for subtle highlights or secondary information.
- Body and headline font: 'Inter', a modern sans-serif, providing a clean, objective, and highly legible aesthetic suitable for financial information and digital interfaces.
- Use crisp, minimal iconography, such as gift boxes, money bags, checkmarks, and user icons, consistent with the clean digital theme, inspired by the '🎁💰' and badge icons in the provided code.
- Centralized, responsive card-based layout to focus user attention on the offer, ensuring readability and accessibility across various screen sizes.
- Subtle UI animations such as smooth fades for content appearance, floating effects for key icons, and gentle button hover/active transitions to enhance interactivity and a polished feel.
